== Description ==

This plugin allows you to restrict users from logging in using IP Address.

== Installation ==

1. Upload `login-ip-restriction.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Users > All Users > Choose user > User IP Address Access Restriction
